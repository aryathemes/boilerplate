# PHP Boilerplate

Boilerplate for new PHP library.

## Requirements

* PHP version 7.1 or greater
* PHPUnit version 7.4 or greater

## Composer

    composer create-project --no-install --remove-vcs luisalberto/php-boilerplate DIRECTORY dev-master
